
AJUSTES — SIMULADOR
1. Logo da Venture:
Inserir o logo da Venture no centro do simulador, em destaque.
2. Texto informativo de tempo:
Adicionar uma mensagem informativa ao usuário indicando que a análise leva menos de 5 minutos para ser concluída.
3. Chamada principal (headline):
O título/chamada do simulador deve ser: "Calcule quanto sua fazenda pode receber com Carbono"
4. Ordem dos campos — Upload do KML antes dos Hectares:
A seção de inserção do arquivo KML deve vir antes da seção de inserção de hectares. O objetivo é forçar o usuário a fazer o upload do KML primeiro. Quando o arquivo KML for carregado, o mapa deve centralizar e ir automaticamente para o polígono criado a partir desse arquivo.
5. Menu lateral — Estimativa de ganho:
Quando o menu vai para a lateral, a estimativa de ganho aparece no final. O ajuste é fazer essa estimativa aparecer antes ou ao lado do seletor de práticas, para que o seletor de práticas e a estimativa de ganhos fiquem visíveis na mesma tela ao mesmo tempo.
6. Composição anual — Scroll:
A parte de composição anual não está rolando para baixo. Acredita-se que essa seja a parte que faltava do comparativo entre manejo regenerativo versus manejo convencional. Esse bug/comportamento precisa ser corrigido para que o conteúdo seja acessível.

AJUSTES — FLUXO DO PARCEIRO REPRESENTANTE COMERCIAL
1. Status geral:
O fluxo está funcionando corretamente. O único ponto pendente é a falta de uma opção para ajustar o nível de comissionamento de forma individual por parceiro.
2. Comissionamento individual por parceiro:
Precisa ser criada uma opção que permita definir o nível de comissão de forma individual para cada parceiro. Ou seja, o Parceiro A poderá ter uma comissão de X% e o Parceiro B terá uma comissão de Y%, de forma independente.
3. Visualização da comissão aplicada:
Incluir a visualização da comissão aplicada dentro da seção "Como funciona a comissão", tanto no nível do parceiro quanto na visão geral. Esse bloco deve mostrar claramente o percentual de comissão que aquele parceiro específico terá.
4. Posicionamento dos níveis de comissão:
O bloco com a visualização dos níveis de comissionamento deve ser movido para cima, posicionando-se antes da linha "Sua posição atual".
5. Área mínima:
Deixar evidente que a área mínima aceita é de 500 hectares.

AJUSTES — FLUXO DO PRODUTOR (CLIENTE)
1. Seletor de anos:
O seletor de anos deve exibir apenas anos únicos, sem repetição — por exemplo: 2026, 2025, 2024, e assim por diante em ordem decrescente.
2. Seletor de dieta predominante:
As opções do seletor de dieta predominante devem ser: Pasto, Silagem/Volumoso, Concentrado e Outro. Quando o usuário marcar a opção "Outro", um campo de texto aberto deve aparecer para que ele digite a dieta. Além disso, o seletor deve permitir marcar mais de uma opção ao mesmo tempo.
3. Campo "Outro adubo":
Na lista de adubos, incluir uma opção chamada "Outro adubo". Quando essa opção for marcada, um campo de digitação aberto deve aparecer para que o usuário escreva qual é o adubo utilizado.
4. Nova seção — Produtos Biológicos:
Criar uma seção chamada "Produtos Biológicos" e posicioná-la antes da seção de fertilizantes orgânicos. O campo do produto dentro dessa seção deve ser um campo de digitação livre (não um campo fixo/fechado).
5. Destaque para anexo de documentos comprobatórios:
Em todos os pontos do fluxo onde seja necessário anexar algum documento, deixar em evidência que o usuário precisa anexar o arquivo comprobatório correspondente.
6. Renomeação de campo — Laudos de Solo:
O botão ou label "Anexar Nota Fiscal" deve ser renomeado para "Laudos de Solo (agricultura de precisão)".
7. Nova seção — Documentos Fundiários:
Criar uma nova seção chamada "Documentos Fundiários", destinada ao upload de documentos relacionados à terra. Os documentos que devem poder ser anexados nessa seção incluem: matrícula atualizada, CAR, GEO, CCIR, contratos de arrendamento, entre outros documentos fundiários relevantes.
8. Nova seção — Plantas de Cobertura:
Incluir uma seção específica para "Plantas de Cobertura". Os campos dessa seção devem ser apenas três: Tipo de cobertura (campo de digitação livre), Data de Semeadura e Data de Dessecação/Rolagem. Além disso, deve haver a possibilidade de o usuário adicionar a nota fiscal de compra de sementes nessa seção também.
